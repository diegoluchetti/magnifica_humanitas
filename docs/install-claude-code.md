1. Copy `skills/guiding-with-magnifica-humanitas/` into your Claude Code skills directory (commonly `~/.claude/skills/` or a project-level skills path your team uses).

3. When a task matches the skill’s `description` triggers, invoke the platform **Skill** mechanism for `guiding-with-magnifica-humanitas` (do not paste ad hoc excerpts as a substitute for loading the whole `SKILL.md`).
