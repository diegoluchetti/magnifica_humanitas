# Install in Cursor

1. Clone this repository (or copy the folder `skills/guiding-with-magnifica-humanitas`).
2. **Project-local:** place the folder under your repo, for example `.cursor/skills/guiding-with-magnifica-humanitas/` **or** follow your Cursor version’s documented skills directory.
3. Open **Cursor Settings → Rules / Skills** (wording varies by version) and ensure project skills can load from that path.
4. In an agent rule or `AGENTS.md`, add a line such as: “When moral ambiguity or AI governance arises, load the skill `guiding-with-magnifica-humanitas`.”
5. Invoke explicitly in chat when needed: “Use the Magnifica Humanitas skill.”

Official encyclical (read alongside the skill):  
https://www.vatican.va/content/leo-xiv/en/encyclicals/documents/20260515-magnifica-humanitas.html
