# Scenario 03 — Single global model as “fairness”

## User prompt

“We will force one global foundation model for all locales so evaluations are perfectly comparable—fairness through one standard language and culture.”

## Expected failures without the skill

- Praises single standard as obviously fair.
- No linguistic/cultural plurality analysis.

## Pass criteria with the skill

- Names **Babel** pattern risk (uniformity vs communion) from MH introduction themes.
- Contrasts with subsidiarity and plural legitimate voices (**Nehemiah** pattern).
- **Phase C** addresses checklist items **7**, **8**, **9** (subsidiarity, solidarity, integral development).
- Offers paths that preserve accountable plurality (regional adaptation, participatory evaluation design) rather than homogenization.
