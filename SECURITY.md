# Security Policy

## Reporting a vulnerability

If you find a security issue in the validation script or repository packaging, please open a private report through your hosting platform's security advisory feature if available.

Do not publish exploit details before maintainers have had a reasonable chance to respond.

## Scope

This repository contains documentation and a standard-library validation script. It does not provide a hosted service or collect user data.

## Safety concerns

If the skill appears to enable exploitation, manipulation, dehumanization, or automated harm, treat that as a safety defect. Open an issue with:

- the prompt or scenario;
- the agent response;
- the Magnifica Humanitas law gate that failed;
- the proposed wording change or new pressure scenario.
