# Examples

These examples show how the `discerning-ai-with-magnifica-humanitas` skill should guide an agent from a risky AI request toward a humane decision.

Each example follows the same structure:

1. User request.
2. Socratic questions about intention, bias, affected persons, power, and safeguards.
3. Magnifica Humanitas LAW gates.
4. Decision: proceed, constrain, or refuse.
5. Humane implementation path.
