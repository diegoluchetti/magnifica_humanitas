# Contributing

Thank you for helping refine this skill.

1. **Do not** paste large excerpts of *Magnifica Humanitas* into the repo; **summarize** and **link** the Vatican URL.
2. Any change to `SKILL.md` should update at least **two** `tests/scenarios/` files or add a new scenario that would have caught a regression.
3. Run `bash scripts/validate-skill.sh` before opening a pull request.
4. For theological accuracy, prefer citations to the official English (or Latin) text on vatican.va; note paragraph numbers when possible.

## Code of conduct

Be charitable, precise, and welcoming. Disagreement about prudential applications is normal; personal attacks are not.

## Security

This repository contains no executable dependencies. If you find a problem with GitHub settings or workflows, report privately to the maintainers of your fork.
